    注释的内容跟JAVA应该是类似的。
    // 单行注释
    /* 多行注释*/
    js 变量的使用可以不需要声明，但是声明变量是一个好习惯，不过跟强类型的语言不同（JAVA..），JAVA中每一个变量都需要指定确定的变量类型，而js中只需要一个 var表明是一个变量即可。js 在被解释的过程中会自动确定变量的具体类型。

    alert(string),弹出框
    数组 var demo = Array(4);demo[0] = "111";or  var demo = Array("111","222");or var demo = ["111","222"];

    获取相应的节点，通过document.  id:getElementById(id名)。
    通过标签：getElementByTagName（标签名）。通过类名：getElementByClassName（对象名）。
    获取到节点之后，可以通过 tempVar.getAttribute(属性名),获取到相应的属性。有get自然就有set 参数是属性名，值。
    点击事件处理，onclick = "functionName;",如果要防止用户被带到目标的链接窗口，调用方法之后，return false.
    js是一种解释型语言，跟编译型语言不同。
    编译型语言需要一个编译器，是一个软件，编译器负责将源代码翻译成可以直接在计算机上执行的文件。编译型的代码出现错误的时候，可以直接在代码编译阶段就能被发现。
    解释型语言需要的是解释器，浏览器中的JS解释器就可以直接读入源代码并且执行，不同的是，解释型语言智能等到解释器执行到有关代码的时候才能发现。很皮

    获取元素偏移的到顶部的距离，如果和当前滚动的距离相等或者小于
    $(function () {
        $(window).scroll(function() {
            var top1 = $('.f_d').offset().top;
            var gun = $(this).scrollTop();
            var top = top1-gun;
            if (top <= 0) {
                $('.f_d').addClass('fixed_top');
            }else{
                $('.f_d').removeClass('fixed_top');
            }
        });
    });
